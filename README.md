# Godot Tools — GridEx Plugin

## Godot Tools is a plugin for GridEx that adds additional tools to accelerate Level Map development inside the Godot Engine.

The plugin is fully compatible with Godot 4.x and is designed to integrate directly into the GridEx workflow within the editor.

# Features
.tscn Export

Export maps directly to Godot’s native scene format (.tscn), ready to be used in your project.

# Optimized Export Pipeline

A faster and more streamlined export process integrated into the editor workflow.

# Scene Wrapper System

A scene encapsulation system that improves organization, reuse, and instancing of tiles and objects.

# Collision Manipulation

Tools for creating and adjusting collisions directly within GridEx.

# Compatibility

GridEx 0.6.x
Godot 4.x
